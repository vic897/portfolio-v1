# 🚀 Modern Portfolio 2026 | Vinay Mathpati

A high-performance, bento-style portfolio built with a "Vibe Coding" approach—leveraging **Next.js 15**, **Tailwind CSS**, and **AI-assisted development (Cursor/v0)** to ship clean, professional UI at speed.

---

## 🛠️ The Tech Stack

| Category | Tools |
| :--- | :--- |
| **Framework** | Next.js 15 (App Router) |
| **Styling** | Tailwind CSS (Glassmorphism & Bento Grids) |
| **Backend** | Next.js Server Actions |
| **Email** | Resend API Integration |
| **Workflow** | Cursor AI + v0.dev (Vibe Coding) |
| **Hardware** | Built on HP Victus (i5 13th Gen / RTX 4050) |

---

## 🌟 Key Features

### 📨 Full-Stack Contact Form
A fully functional contact system using **Server Actions**. No client-side API exposure. 
* Includes a custom `SubmitButton` with real-time loading states via `useFormStatus`.
* Direct-to-inbox delivery with automated `replyTo` logic for seamless networking.

### 🎨 Design Philosophy
* **Bento Grid Layout:** Organized, mobile-first project showcases.
* **Performance First:** Optimized for Core Web Vitals (LCP/CLS) using the Next.js `<Image />` component.
* **Accessibility:** Semantic HTML and ARIA-ready components.

---

## 📱 Parallel Project: Android System Design
While building this web presence, I am simultaneously exploring **AOSP and Kernel modding** on legacy hardware (Samsung J5 Prime). 
* **Current Focus:** Analyzing partition structures and `build.prop` optimization using ADB via WSL2.
* **Goal:** Building a custom, lightweight ROM for performance-constrained devices.

## 🚀 Getting Started

1. **Clone the repo:**
   ```bash
   git clone [https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git)
````

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Set up Environment Variables:**
    Create a `.env.local` file and add your Resend API key:

    ```text
    RESEND_API_KEY=re_your_key_here
    ```

4.  **Run the dev server:**

    ```bash
    npm run dev
    ```
